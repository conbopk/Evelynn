# Evelynn-Inspired Favicons
## AI Image Generation App

### Variants
| Prefix    | Inspiration          | Best use                          |
|-----------|----------------------|-----------------------------------|
| heart     | W - Allure (heart)   | Primary; cleanest at 16px         |
| spike     | Q - Hate Spike       | Sharp/dark; suits bold dark UIs   |
| whiplash  | E - Whiplash         | Swirling energy; loader/splash    |
| caress    | R - Last Caress      | Silhouette; app icon / splash     |
| ai_eye    | Composite AI eye     | Modern AI aesthetic; strong 32px+ |

### Sizes per variant: 16 / 32 / 48 / 64 / 128 / 180 / 192 / 512 px + .ico

### Drop-in files
- favicon.ico                (heart, 16/32/48 multi-size ICO)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png       (180x180, iOS home screen)
- android-chrome-192x192.png
- android-chrome-512x512.png
- site.webmanifest
